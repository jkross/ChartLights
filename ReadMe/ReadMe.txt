This directory contains a description of the project and a "making of" document.

Other project directories are as follows:
	Arduino				Source Code for the Arduino project which drives the LEDs
	Eagle				Design files for the custom PCB
	SketchUp			3d design files for the custom 
							* Arduino Uno mounting base
							* LED holders
	LightList			Downloaded light list from USCG & scripts to extract and sort the lights
							For all lights within the chart bounding box, sort first on 
							the brightest lights, and then on the lights closest to the Driftmier home
							or the Driftmier boat.
